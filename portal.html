Client and Staff Portal









Welcome to the client and staff portal! Please choose the best option.

Please note: If you are registering for the first time, please register prior to selecting other options.

Staffing Request


Clients can use this section to start a staffing request. After submission, clients will be provided an ID which is used to access the staffing request information for editing staff requests.

Staffing Request Information

Clients can view or modify current staffing requests associated with the Request ID.

Staffing Administration


Root users or manager access only.

Contract Information


Root users or manager access only. Allows users to view, upload, or delete contracts.

Edit Profile


Staff only. Staff can update information such as resume, address, salary, education, and picture.

Client Information

Clients can view resumes, pictures, location, education, and desired salary of potential candidates. Teachers: Please register your students to add them to your rosters.

User Access Information

Authorized users only. Can add new users and view, modify or delete any existing users in database for added security.
